# Technologies

UDIE leverages a modern, full-stack technologies focused on performance and spatial awareness.

## Frontend (Mobile)
- **Language**: Swift 5.10
- **Framework**: SwiftUI
- **Maps**: Apple MapKit
- **Networking**: URLSession + Async/Await
- **Reactivity**: Combine

## Backend (Server)
- **Language**: TypeScript (Node.js)
- **Framework**: NestJS
- **Database**: PostgreSQL 16
- **Spatial Engine**: PostGIS 3.4
- **Cache**: Redis 7 (Alpine)
- **Validation**: class-validator + class-transformer
- **Linting**: ESLint 9 + `@typescript-eslint`
- **Testing**: Node test runner (`node --test`) for service-level checks

## Infrastructure
- **Containerization**: Docker & Docker Compose
- **Scripting**: Shell (Bash/Zsh)
- **Version Control**: Git

## Development Tools
- **IDE**: Xcode, VS Code
- **API Testing**: Postman, curl
- **Documentation**: Mermaid.js, Markdown
- **Build Verification**: `xcodebuild`, `nest build`, Docker Compose
